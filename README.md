# **Ywork** | NodeJS Web FrameWork
##
#### Install:
- `npm install`
#### Dev Mode Http Server:
- `npm run sv`
#### Dev Mode WebSocket Server:
- `npm run ws`
#### Deploy:
- `npm start`
